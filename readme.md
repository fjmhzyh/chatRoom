nodejs,express,socket.io搭建小型聊天室

### 项目搭建
```bash
git clone git@github.com:fjmhzyh/react-china.git      // 将项目下载到本地
$ node server.js        // 启动项目
```
* 打开项目主页  http://localhost:3000
